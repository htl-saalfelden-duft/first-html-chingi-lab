Bild und Text

CSS steht für Cascading Style Sheets (kaskadierende Stilvorlagen) und ist eine Stylesheet-Sprache, die das Aussehen und Layout von Webinhalten, typischerweise HTML-Dokumenten, gestaltet. CSS trennt den Inhalt (HTML) von der Darstellung (CSS), sodass man Farben, Schriftarten, Positionen und Responsive Designs für verschiedene Bildschirme festlegen kann. Es ist ein sich entwickelnder Standard, der von vielen Webseiten genutzt wird, um ein einheitliches und ansprechendes Erscheinungsbild zu erzeugen.
